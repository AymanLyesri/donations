Donation Widget Demo - Dark Theme

This page demonstrates the donation widget in a dark-themed environment.

The widget automatically detects the dark theme and adjusts its appearance.

API Methods

You can also trigger the widget programmatically:

Embed Code

Add this to your website:

<script src="http://localhost:3000/widget.js"></script>

Theme Detection

The widget detects dark mode by:

  • Checking for dark class on html/body elements
  • Reading prefers-color-scheme media query
  • Live updates via MutationObserver
